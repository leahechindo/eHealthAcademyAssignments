Understanding the Web and its Terms

I am new to the world of web development and a month ago, I had no idea what to do or how to go about it.
The first thing I was taught on the elearning platform was how the web works (the behind the scenes of what a browser presnts).
I loved the classes and how it was so easy to understand, I have never really thought of being a web developer and this is still quite exciting for me.
